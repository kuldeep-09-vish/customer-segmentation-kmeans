# Customer Segmentation using KMeans Clustering

Machine Learning project using Unsupervised Learning (KMeans Clustering) to segment customers based on demographics, income, spending behavior, and purchase patterns.

## Features
- Customer segmentation using KMeans
- Data preprocessing pipeline
- Streamlit web application
- Real-time cluster prediction

## Technologies
- Python
- Pandas
- NumPy
- Scikit-Learn
- Streamlit
- Joblib
